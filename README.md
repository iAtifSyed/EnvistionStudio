## ✨ Overview

**Envision Studio** is a world-class, premium, highly responsive HTML5 template designed for technology companies, software agencies, and digital transformation consulting firms. Built entirely with pure HTML5, modern CSS3 variables, and high-performance Vanilla JavaScript—**zero heavy external dependencies**.

> 💡 **Perfect for:** Tech startups, SaaS platforms, IT consultancies, digital agencies, and software developers looking for a stunning, ready-to-launch website.

## 🎨 Visual Quality & Design Architecture

### 🎯 Design Philosophy
- **Thematic Balance**: Generous whitespace, high-contrast dark/light integrated schemas, and modern glassmorphism backdrops
- **Pixel-Perfect**: Every element meticulously crafted for visual harmony
- **Accessibility-First**: WCAG-compliant contrast ratios and semantic HTML

### 🎭 Color Palette
| Role | Light Mode | Dark Mode |
|------|------------|-----------|
| **Primary** | `#2563EB` | `#2563EB` |
| **Secondary** | `#06B6D4` | `#06B6D4` |
| **Accent** | `#8B5CF6` | `#8B5CF6` |
| **Background** | `#FFFFFF` / `#F8FAFC` | `#0F172A` / `#020617` |
| **Text** | `#1E293B` | `#F8FAFC` |

### 🔤 Typography
- **Display:** `Space Grotesk` — Bold, modern headlines
- **Body:** `Inter` — Ultra-legible, pixel-perfect sans-serif
- **Code:** `JetBrains Mono` — For code snippets and status badges

### 🖼️ Icon System
- **100% Standalone**: Vector-based inline SVGs embedded directly in markup
- **Zero Latency**: No external CDN requests, loads instantly
- **Offline Ready**: Fully functional without internet connection

## ⚡ Interactivity & Animations

Our Vanilla JavaScript engine powers **8 premium interactive features**:

### 🎬 Core Features
1. **Preloader Screen** — Smooth fade transition on load
2. **Glassmorphic Sticky Header** — Shrinks with blur/border on scroll
3. **Double-Action Dark Mode** — Toggle with localStorage persistence
4. **Scroll Reveal Animations** — IntersectionObserver-powered fade/slide
5. **Real-time Stat Counters** — Easing progress animations from 0 to target
6. **Case Study Filter** — Instant filtering with scaling/fading transitions
7. **Accordion FAQs** — Auto-calculated height with adjacent closing
8. **Contact Validation** — Regex email validation with loading states & alerts

## 🛠️ Execution & Deployment

### 🔧 Mode A: Standalone (No tools needed)
```bash
# Simply double-click any .html file
# or open in any modern browser

# All files work offline, no server required!
```

### 🚀 Mode B: Optimized Dev Server (Vite)
```bash
# Prerequisites: Node.js installed

# 1️⃣ Install dependencies
npm install

# 2️⃣ Start development server (auto-refresh)
npm run dev

# 3️⃣ Build production bundles
npm run build

# 4️⃣ Preview production build
npm run preview
```
